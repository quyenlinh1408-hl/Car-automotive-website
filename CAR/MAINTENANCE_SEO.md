# Bao Tri Va Nang Cap Web Car Automotive

## 1. Viec can lam sau moi lan cap nhat
- Kiem tra lai cac trang: car.html, feedback.html tren mobile va desktop.
- Kiem tra JS console, dam bao khong co loi.
- Chay lai nhanh hieu nang (load time, anh nang, so request).
- Neu sua URL hoac them trang moi, cap nhat sitemap.xml.

## 2. Checklist de len Google on dinh
- Deploy website len domain that su (khong dung localhost).
- Vao Google Search Console, them property domain.
- Xac minh quyen so huu domain.
- Gui sitemap: https://quyenlinhfptauto.vn/sitemap.xml
- Yeu cau index lai URL chinh:
  - https://quyenlinhfptauto.vn/car.html
  - https://quyenlinhfptauto.vn/feedback.html
- Theo doi Coverage va Core Web Vitals hang tuan.

## 3. Nhung file SEO da co trong project
- robots.txt
- sitemap.xml
- Meta SEO + Open Graph trong car.html va feedback.html
- Structured data JSON-LD trong car.html

## 4. Cau hinh domain hien tai
- Domain dang duoc gan: https://quyenlinhfptauto.vn
- Neu doi domain khac trong tuong lai, cap nhat:
  - sitemap.xml
  - robots.txt
  - JSON-LD trong car.html
  - canonical va og:url trong car.html, feedback.html, index.html
- Neu co logo chuan thuong hieu, bo sung og:image chat luong cao 1200x630.
